Je veux modifier complètement la fonctionnalité “Cagnotte” pour supprimer Stripe et passer à un système de contribution multi-moyens externes.

Objectif :
Simplifier l’expérience utilisateur et permettre aux invités de contribuer via PayPal, Mobile Money, lien externe ou virement.

-----------------------------------
1. SUPPRESSION STRIPE
-----------------------------------

- Supprimer Stripe de la cagnotte
- Supprimer toute logique de paiement intégré
- Garder uniquement des moyens externes

-----------------------------------
2. BACKOFFICE
-----------------------------------

Créer une section :
“Moyens de contribution”

Types disponibles :
- PayPal (email)
- Téléphone (Mobile Money)
- Lien externe
- Virement bancaire

Chaque moyen a ses champs :

PayPal :
- email

Téléphone :
- numéro
- label (ex : Wave, Orange Money)

Lien :
- URL
- nom du service

Virement :
- IBAN
- nom titulaire
- banque

Permettre :
- ajouter
- supprimer
- activer/désactiver
- réordonner

-----------------------------------
3. FRONT (SITE PUBLIC)
-----------------------------------

Afficher les moyens sous forme de cartes :

Chaque carte contient :
- icône
- titre
- description
- bouton d’action

Actions :

PayPal → redirection
Lien → redirection
Téléphone → afficher + bouton “Copier”
Virement → afficher + bouton “Copier”

-----------------------------------
4. CONTRIBUTION MANUELLE
-----------------------------------

Ajouter un bouton :
“J’ai contribué”

Quand l’utilisateur clique :
ouvrir un formulaire :

- Nom
- Montant
- Message

Enregistrer la contribution

-----------------------------------
5. INTÉGRATION EXISTANTE
-----------------------------------

Les contributions doivent :
- apparaître dans la liste des contributions
- déclencher les animations live (confettis)
- fonctionner avec SSE existant

-----------------------------------
6. UX
-----------------------------------

- design premium
- mobile-first
- simple
- rassurant

-----------------------------------
7. LIVRABLES
-----------------------------------

- suppression Stripe
- nouveau système multi-moyens
- contribution manuelle
- UI backoffice + front
