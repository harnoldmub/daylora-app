2️⃣ TRANSFORMATION DU MODE ÉDITION
A. Ajouter un vrai bandeau d’édition flottant

En bas ou en haut :

Card flottante glassmorphism :

Blur background

Border subtle

Rounded-full

Shadow-xl soft

Contenu :

🟢 Mode édition actif
Bouton “Terminer” (primary)
Bouton “Aperçu” (secondary)

Ne plus afficher juste un bouton simple.

B. Encadrer les zones éditables proprement

Quand hover sur une zone éditable :

border dashed subtle rose/beige

fond très léger beige transparent

icône ✏️ qui apparaît en haut à droite

label discret : “Modifier”

Ne pas afficher les vrais inputs tant que l’utilisateur ne clique pas.

C. Edition inline premium

Quand on clique :

Transformer le texte en input élégant

Input sans bordure visible

Background soft beige

Auto focus

Bouton ✓ et ✕ discrets à droite

Animation :

fade + scale 0.98 → 1

150ms smooth

D. Supprimer le style brut des inputs

Actuellement :

bordures trop visibles

couleur trop forte

trop administratif

Remplacer par :

border-transparent

focus:ring-2 ring-rose-300

shadow-inner subtle

rounded-xl

E. Uniformiser tous les champs édition

Règle :

• même border-radius partout (16px)
• mêmes ombres
• même padding
• mêmes transitions
