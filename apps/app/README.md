# Libala Vitrine

This directory contains the frontend application for Libala.

## Setup

1.  Navigate to this directory:
    ```bash
    cd vitrine
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

## Development

To run the frontend in development mode:

```bash
npm run dev
```

The frontend is configured to proxy API requests to the backend server running at `http://localhost:5000`. Ensure the backend is running in the root directory.

## Build

To build for production:

```bash
npm run build
```
